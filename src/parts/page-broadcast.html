▶ One-time setup: deploy the send-notification edge function
To actually send emails, you need to deploy a Supabase Edge Function that connects to an email provider (recommended: Resend — free tier covers 3,000 emails/month and setup is minimal).
  1. Create an account at resend.com and verify your sending domain (or use their onboarding domain for testing).
  2. Copy your Resend API key.
  3. In your Supabase dashboard → Edge Functions → Create function → name it send-notification.
  4. Paste the Deno code below.
  5. In the function's Secrets, add:
    RESEND_API_KEY = your Resend API key
    FROM_EMAIL = e.g. scheduler@yourdomain.com (must be a verified sender)
  6. Deploy the function.
