🖥 Desktop Widget
Generate pairing codes that physicians paste into the RadScheduler desktop widget (macOS + Windows). The widget shows their wRVU goal, scheduled shifts, study count, and drive-time credit for the day. Future PACS integration plugs in via the widget's API hook.
📦 Widget download URLs
Paste the URLs of your published widget installers (typically a GitHub Releases asset). The "Send install kit" flow below uses these to give physicians one-click downloads. Build the widget once with ./widget/build-mac.sh + ./widget/build-win.cmd and either:
  • Upload to GitHub Releases (free, no infra) and paste the asset URLs
  • Or upload to your own file host (Dropbox, S3, intranet) and paste those URLs
URLs are stored in S.cfg and embedded into every install-kit email — physicians click straight from their inbox.
🚀 Send install kit (recommended)
One-click install kit per physician: bundles the OS-specific download link with a pre-paired code. Physician copies the code, downloads, installs, launches — widget auto-pairs from their clipboard, no paste required.
